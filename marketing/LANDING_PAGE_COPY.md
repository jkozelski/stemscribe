## 10. FAQ Section

### Section Header
```
QUESTIONS
```

**Q: How accurate are the tabs?**
```
A: Drums and piano are our strongest instruments — the custom neural
networks handle them well. Guitar and bass are good but not perfect,
especially for fast or heavily distorted passages. Think of the output
as a strong first draft that saves you 80% of the transcription work.
We're honest about this because we're musicians too.
```

**Q: What audio formats do you support?**
```
A: MP3, WAV, FLAC, OGG, M4A, and most common audio formats. For URLs,
we support YouTube, Spotify, Apple Music, SoundCloud, Bandcamp, and
Archive.org. Max file size is 500MB.
```

**Q: Can I process a full album?**
```
A: Yes — upload tracks one at a time. Each one gets its own set of stems,
MIDI files, and Guitar Pro tabs. On the Pro plan, there's no limit to
how many songs you can process per month.
```

**Q: What about copyright?**
```
A: StemScribe is a tool for personal learning and practice, similar to
slowing down a record or transcribing by ear. We don't host or distribute
copyrighted content. Your processed files are stored in your private
library and not shared with anyone.
```

**Q: How long does processing take?**
```
A: A typical 4-minute song takes 2-5 minutes to separate and transcribe.
Pro users get priority queue access for faster processing during peak times.
```

**Q: Can I edit the generated tabs?**
```
A: The Guitar Pro (.gp5) files open in Guitar Pro, TuxGuitar (free), or
any compatible editor. Edit notes, fix fret positions, add dynamics —
it's your file. MIDI exports work in any DAW.
```

**Q: Do you have a mobile app?**
```
A: Not yet. StemScribe is web-based and works on any device with a browser.
A mobile app is on our roadmap.
```

**Q: What if the tabs are wrong?**
```
A: AI transcription is good but not infallible. The Guitar Pro export is
designed for you to review and correct. We find it saves 70-80% of the
manual transcription effort. We're constantly improving the models.
```

## Conversion Funnel

### Stage 1: Awareness (Top of Funnel)
**Goal:** Get musicians to the landing page
**Channels:** Reddit organic posts, YouTube SEO videos, Show HN, Product Hunt, TikTok stem separation demos
**Key metric:** Unique visitors

### Stage 2: Interest (Landing Page)
**Goal:** Scroll past the hero, understand the value prop
**Friction points to eliminate:**
- No account required for first use (critical — reduces friction to zero)
- Show real output examples (screenshot of Guitar Pro tabs, audio player with stems)
- Comparison table makes the value obvious vs. stitching together multiple tools
**Key metric:** Scroll depth, time on page

### Stage 3: Activation (First Song Processed)
**Goal:** User uploads a song they care about and gets results
**Tactics:**
- "Try It Free" CTA goes directly to the upload interface, not a signup page
- Pre-loaded demo songs (Tidepool Artists tracks) for users who want to try without uploading
- Processing progress screen shows what's happening ("Separating stems... Transcribing guitar... Building Guitar Pro file...")
- Results page auto-plays the original with stems visible — instant "wow" moment
**Key metric:** Songs processed (north star metric)

### Stage 4: Revenue (Free → Paid Conversion)
**Goal:** Convert free users who hit limits to Premium or Pro
**Triggers:**
- Free tier limit (3 songs/month) — show upgrade prompt after 3rd song: "You've used all 3 free songs this month. Upgrade to Premium for 50 songs/mo at $4.99."
- Feature gate — when a free user tries to export Guitar Pro: "Guitar Pro export is a Pro feature. Upgrade for $14.99/mo to download .gp5 files."
- Feature gate — when a free user hits the 5-minute duration limit: "Songs over 5 minutes need Premium. Upgrade for $4.99/mo."
- Quality gate — show preview of 6-stem separation quality vs. free tier's 4-stem
**Key metric:** Free → Paid conversion rate (target: 3-5%)

### Stage 5: Retention
**Goal:** Keep paid users processing songs monthly
**Tactics:**
- Song library persists — every processed song is accessible anytime
- Email digest (monthly): "You processed 12 songs this month. Here's your library."
- New feature announcements that give users a reason to come back
- Practice mode creates daily habit loops (users return to practice, not just transcribe)
**Key metric:** Monthly active users, songs processed per user per month

### Stage 6: Referral
**Goal:** Users recommend StemScribe to bandmates and music friends
**Tactics:**
- Shareable Guitar Pro files — when a user shares a GP file, it's a natural product demo
- "Processed with StemScribe" watermark option on MIDI exports (opt-out for Pro users)
- Referral program: Give a friend 5 extra free songs, get 5 extra yourself
- Band sharing: Pro users can invite bandmates to view (not download) their library
**Key metric:** Referral signups, viral coefficient
