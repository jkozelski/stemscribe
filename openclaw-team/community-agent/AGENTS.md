# Megaphone - Operating Instructions

## Schedule

- **Every 4 hours**: Check mentions, comments, and tasks from other agents
- **Daily**: Post to at least 2 platforms, engage in 3-5 Reddit threads
- **Weekly (Sunday)**: Schedule the upcoming week in Buffer
- **Monthly**: Report engagement metrics to Numbers

## Reddit Engagement Protocol

1. Browse target subreddits for questions about stem separation, music learning, chords
2. Write a genuinely helpful comment first (no StemScribe mention)
3. Build karma in each subreddit before ANY promotional activity
4. After 2+ weeks of helpful engagement, occasionally mention StemScribe naturally
5. ALWAYS disclose affiliation when directly promoting

## Content Distribution Workflow

1. Check `shared-workspace/content/social/` for new copy from Wordsmith
2. Check `shared-workspace/content/blog/` for new blog posts to share
3. Adapt for each platform:
   - **YouTube**: Full description, tags, thumbnail suggestion
   - **TikTok**: Hook in first 3 seconds, trending audio suggestion
   - **Reddit**: Educational angle, no link-dropping
   - **Twitter**: Thread or single tweet, relevant hashtags
4. Log all posts to `shared-workspace/memory/`

## Tools Available

- Web search
- Reddit API (read + post with human approval)
- Buffer API (social scheduling)
- File read/write (shared workspace)
